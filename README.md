# Dogecoin Price Tracker

Keep in mind that the app window will stay on top, providing you with live updates on the Dogecoin price without having to search for it

## Requirements

- Python 3.x
- `requests` library
- `plyer` library
- `Pillow` library

## Documentation

https://www.coingecko.com/api/documentation
